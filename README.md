# AndroidRapidDevelopmentFramework
#### 安卓快速上手开发基础框架 ,包含UI框架、网络框架等。主要包括MVP 、Kotlin、Retrofit2、Rxjava2、md风格 ;
 
 ##### 简介 
    com.rapid_development.framework
        - base : Activity、fragment、dialog 等基类 ;
        - data : 网络库、SP、Log等数据相关类 ;
        - extends : 一系列扩展函数 ;
        - ui :UI 及业务逻辑相关 ;
        - utils : 一些工具类 ;
